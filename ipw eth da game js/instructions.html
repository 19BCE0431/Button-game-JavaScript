Game Instructions

  1. Press a key to start the game.
  2. Observe the button that has been clicked. You can also use the help of the unique sound that plays.
  3. To win the level, try cliking on the same button.
  4. On CORRECT guess, you'll be moved to next level.
  5. On WRONG guess, you'll have to press any key to restart the game from level 1.
